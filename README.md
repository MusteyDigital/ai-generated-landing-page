# Tideline — AI-Generated Landing Page

A responsive one-page marketing site for **Tideline**, a fictional weekly
planning app for remote teams. Built with semantic HTML5 and CSS Grid/Flexbox.

## Sections
- Hero — headline, subtext, and a primary call-to-action button.
- Features — three cards (Flow / Tides / Drift) laid out in a CSS Grid
  that stacks to one column on mobile.
- CTA — call-to-action band inviting sign-up.
- Footer — logo, nav links, copyright.

## How AI was used
AI helped with planning and copy only:
- Suggested the app name "Tideline" and the tide/rhythm metaphor.
- Drafted the hero headline/subtext and the three feature blurbs.
- Suggested the color palette (ink navy, warm paper, coral, sage).

All HTML structure and CSS (layout, Grid/Flexbox, the responsive
media query) were written by hand.
